安工大到梦空间后台管理开放API
发表于 2020-03-30| 更新于 2020-03-30
  • 接口地址: http(s)://dmapi.webpro.ltd

获取某学号所有实时得分详情

  • URL: /dm/getUserInfo/{stuNo}
  • 参数
    参数名称 说明 备注
    stuNo 学号 安工大学生学号
  • 返回值示例
    json
  • 示例：
    • http://dmapi.webpro.ltd/dm/getUserInfo/189074246

获取学院列表

  • URL: /dm/getCollege
  • 参数
    参数名称 说明 备注
  • 返回值示例
    json
  • 示例：
    • http://dmapi.webpro.ltd/dm/getCollege

获取班级列表

  • URL: /dm/getClassByCollegeId/{collegeId}
  • 参数
    参数名称 说明 备注
    collegeId 学院id 学院列表中获取的key就是collegeId
  • 返回值示例
    json
  • 示例：
    • http://dmapi.webpro.ltd/dm/getClassByCollegeId/288

按学院检索

  • URL: /dm/getInfoByCollegeId/{collegeId}/{beginDate}/{endDate}/{page}
  • 参数
    参数名称 说明 备注
    collegeId 学院id
    beginDate 开始日期 格式:YYYY-MM-DD
    endDate 结束日期 格式:YYYY-MM-DD
    page 页码 从1开始，每页1000条数据
  • 返回值示例
    json
  • 示例：
    • http://dmapi.webpro.ltd/dm/getInfoByCollegeId/265/2020-03-20/2020-03-30/1

按班级检索

  • URL: /dm/getInfoByClassName/{collegeId}/{className}/{beginDate}/{endDate}/{page}
  • 参数
    参数名称 说明 备注
    collegeId 学院id
    className 班级名称 注意这个是班级名称,不是班级id
    beginDate 开始日期 格式:YYYY-MM-DD
    endDate 结束日期 格式:YYYY-MM-DD
    page 页码 从1开始，每页1000条数据
  • 返回值示例
    json
  • 示例：
    • http://dmapi.webpro.ltd/dm/getInfoByClassName/265/计d191/2020-03-01/2020-03-30/1

按年级学院检索

  • URL: /dm/getInfoByJoinYearAndCollegeId/{collegeId}/{beginDate}/{endDate}/{joinYear}/{page}
  • 参数
    参数名称 说明 备注
    collegeId 学院id
    beginDate 开始日期 格式:YYYY-MM-DD
    endDate 结束日期 格式:YYYY-MM-DD
    joinYear 年级 例如2017级就传2017
    page 页码 从1开始，每页1000条数据
  • 返回值示例
    json
  • 示例：
    • http://dmapi.webpro.ltd/dm/getInfoByJoinYearAndCollegeId/265/2020-03-01/2020-03-30/2018/1

按年级班级检索

  • URL: /dm/getInfoByJoinYearAndClassName/{collegeId}/{className}/{beginDate}/{endDate}/{joinYear}/{page}
  • 参数
    参数名称 说明 备注
    collegeId 学院id
    className 班级名称 注意这个是班级名称,不是班级id
    beginDate 开始日期 格式:YYYY-MM-DD
    endDate 结束日期 格式:YYYY-MM-DD
    joinYear 年级 例如2017级就传2017
    page 页码 从1开始，每页1000条数据
  • 返回值示例
    json
  • 示例：
    • http://dmapi.webpro.ltd/dm/getInfoByJoinYearAndClassName/265/计171/2020-03-01/2020-03-30/2017/1
