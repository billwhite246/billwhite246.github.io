支付安全问题

Code
  • 答: 抓包就是去截取app发出的数据包，如果截获就有可能修改参数，破解，比如买东西，你付款100元，如果从app发出数据包，黑客篡改100元为1分，成功支付后，100元的订单就会变为已支付状态，因此如果要在客户端直接发送金额到服务器扣款，必须保证客户端是信任的，否则绝对不允许app直接发送扣款金额到服务器。
