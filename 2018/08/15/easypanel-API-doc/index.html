接口概览

Easypanel的api通信安全码在easypanel的服务器设置处设置。

  • 有以下接口接口：
接口名称 用途
add_vh 创建空间和修改空间
update_vh 暂停空间和恢复空间
del_vh 删除空间
change_password 修改空间的密码
