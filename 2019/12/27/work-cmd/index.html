springboot 接收参数，会忽略/#/后的内容

如果参数中要传/#/，需要进行 urlencode(/#/)操作
http://127.0.0.1:8087/my-lms/YiBanAuth/auth?callBack=http://192.168.1.102:8081/#/enter
=>
http://127.0.0.1:8087/my-lms/YiBanAuth/auth?callBack=http%3A//192.168.1.102%3A8080/%23/enter

swagger 测试 bladex 的 save 方法

使用 swagger 测试 bladex 的 save 方法时，swagger 会传 id: 0，需要把这行删除，不然数据库中的 id 字段不会调用 bladex 的 id 自动生成方法，会直接把 0 当作 id 存储。

centos7 配置网卡

bash


ifcfg-en016777736 表示的是 en016777736 网关的配置文件，en016777736 是网关名称，不同的操作系统名称可能不同，只需要操作相关的配置文件即可

bash

  • DHCP 方式获取 IP
    将 ONBOOT=no 修改为 ONBOOT=yes
    systemctl restart network.service

  • 静态 IP
    BOOTPROTO：获取 IP 地址方式
    ONBOOT：开机是否自动打开网卡
    将 BOOTPROTO=dhcp 修改为 BOOTPROTO=static
    将 ONBOOT=no 修改为 ONBOOT=yes
    IPADDR0=192.168.67.129
    PREFIX0=255.255.255.0
    GATEWAY0=192.168.67.1
    DNS1= 114.114.114.114
    DNS2=8.8.8.8
    IPADDR0：IP 地址
    PREFIX0：子网掩码
    GATWAY0：网关地址
    DNS1：主 DNS
    DNS2：备用 DNS
    systemctl restart network.service

没有 ifconfig 命令

bash

通过 yum search 这个命令我们发现 ifconfig 这个命令是在 net-tools.x86_64 这个包里，接下来我们安装这个包就行了,命令：yum install net-tools.x86_64
安装完成后，再次使用 ifconfig 命令就可以查看到所有的网卡了。
